# 📊 AI Retail Sales Analytics Dashboard

An interactive Streamlit dashboard built for analyzing retail sales data with dynamic filters, KPIs, and visual insights.

---

## 🚀 Features

- 📊 Key Performance Indicators (Sales, Profit, Orders, Customers)
- 🎛 Multi-dimensional filters (Region, Category, Segment)
- 📈 Monthly Sales Trend Analysis
- 🌍 Profit by Region
- 🏆 Top Customers & Products Analysis
- 💾 Export filtered data as CSV
- 📊 Interactive Plotly visualizations

---

## 🛠 Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
